# magictrails

A collection of media and memories from various trekking trails, organized by region and trek names.

## Treks Included
- **Buran Ghati**
- **Chandrashila Tungnath**
- **Kuari Pass**
- **Roopkund**

## Contents
- `.webp` converted image assets for fast loading and web compatibility.
- `.mp4` video clips.
